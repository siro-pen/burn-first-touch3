# BurnFlow Pro | 熱傷初期対応支援アプリ

形成外科医・救急医が、救急外来や回診中に片手で素早く熱傷評価と治療方針の決定を行えるように設計されたプログレッシブ・ウェブアプリ（PWA）です。

## 🚀 主な機能

- **Body Mapによる直感的な評価**: 部位をタップするだけでTBSA（熱傷面積）を自動入力。
- **輸液計算 (Parkland法)**: 患者の体重と面積に基づき、最初の24時間の輸液量を即座に算出。
- **重症度判定**: 日本熱傷学会の基準に準拠した重症度分類と転送推奨の自動表示。
- **AI臨床アドバイザー**: Google Gemini 3 Proを搭載。症例に基づいた具体的な処置・外科的介入・ドレッシング材の選択を提案。
- **スマホ最適化**: ボトムナビゲーションとSafeArea対応により、現場での片手操作を実現。

## 🛠 セットアップとデプロイ

### 1. 公開（デプロイ）
このリポジトリを [Vercel](https://vercel.com/) または [Netlify](https://www.netlify.com/) にインポートしてください。

### 2. 環境変数の設定
AI機能を有効にするため、以下の環境変数を設定する必要があります。
- `API_KEY`: [Google AI Studio](https://aistudio.google.com/app/apikey) で発行したAPIキー

## 📱 使い方
1. 公開されたURLにアクセス。
2. **評価タブ**: Body Mapの部位をタップし、年齢・体重を入力。
3. **AI分析ボタン**: タップすると、現在の症例に対する専門的な治療アドバイスが生成されます。
4. **サマリーコピー**: AI分析結果の右上にあるコピーボタンで、電子カルテ等への貼り付け用テキストを取得。
